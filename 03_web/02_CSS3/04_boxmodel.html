CSS3: 박스모델(box-model)과 속성

박스모델의 구성요소

  • content : HTML 태그의 텍스트나 이미지가 출력되는 부분
  • padding : 콘텐츠를 직접 둘러싸고 있는 내부 여백
  • border : 패딩 외부의 테두리
  • margin : 박스의 맨 바깥 영역

박스모델의 속성

  • contents와 관련된 속성 : width, height
  • padding과 관련된 속성: padding, padding-top, padding-bottom, padding-left, padding-right
  • border와 관련된 속성: border 선의 모양, 선 굵기, 선 색상
    • border-width: border-top-width, -bottom-, -left-, -right-
    • border-style: border-top-style, -bottom-, -left-, -right-
    • border-color: border-top-color, -bottom-, -left-, -right-
  • margin과 관련된 속성: margin, margin-top, margin-bottom, margin-left, margin-right

box 모델의 padding과 margin의 값 지정 방식

  • margin : 값 => 위, 오른쪽, 아래, 왼쪽의 여백을 동일한 값으로 설정
  • margin : 값1, 값2 => 위, 아래는 값1로, 오른쪽과 왼쪽은 값2로 설정
  • margin : 값1, 값2, 값3, 값4 => 위, 오른쪽, 아래, 왼쪽의 여백 순으로 설정
