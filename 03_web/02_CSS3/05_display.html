박스모델의 display 속성 제어

display 속성: 태그가 화면에 보여지는 방식(박스유형)을 제어

  • none: 화면에 보이지 않음
  • block: 블록박스 형식으로 지정, margin, padding 속성의 상하좌우를 지정할 수 있음
  • inline: inline 형식으로 지정, margin, padding 속성의 좌우만 지정할 수 있음
  • inline-block: 블록과 인라인의 중간 형식, inline과 같이 배치하나 margin, padding 속성은 상하좌우 모두 지정할 수 있음
